How it works
From raw websites to machine-readable benchmarks
Every data point comes from real websites, analyzed by AI, verified by frequency.
01
Research
Deep analysis of each industry to identify what features matter. Custom checklists per industry.
02
Crawl
Playwright visits 30 sites per market. Takes screenshots of homepage, products, and key flows.
03
Analyze
GPT-4o Vision examines every screenshot and identifies which features are present.
04
Classify
Frequency analysis: 80%+ = CRITICAL. 50-79% = REQUIRED. 25-49% = RECOMMENDED. <25% = OPTIONAL.
The API
One API call. No signup. No API key.
Free, open, and ready to use in your CI/CD pipeline, product audits, or competitive analysis.
# List all 187 industries
curl https://api.benchmarkhq.site/industries

# Get benchmark for any industry
curl https://api.benchmarkhq.site/benchmark/ecommerce_usa

# Check your site against the benchmark
curl -X POST https://api.benchmarkhq.site/check \
  -H "Content-Type: application/json" \
  -d '{"industry":"crm","features":["free_trial","api_docs","sso"]}'
