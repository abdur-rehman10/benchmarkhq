# Contributing to BenchmarkHQ

Thanks for your interest in contributing! Here's how you can help.

## Adding New Industries

1. **Research** — Create a feature checklist for the industry (see `data/` for examples)
2. **Site List** — Compile 30 sites for the industry/country
3. **Scan** — Run the scraper against your site list
4. **Submit** — Open a PR with the data in the appropriate `data/` directory

## Adding New Countries (E-commerce or News)

1. Fork the repo
2. Add sites to the country list in the scraper config
3. Run the scan
4. Submit a PR with screenshots and frequency analysis

## Improving Data Quality

- Report incorrect feature classifications by opening an issue
- Submit corrections with evidence (screenshots, links)
- Suggest new features to add to existing checklists

## Code Contributions

- Bug fixes and improvements to scrapers
- API enhancements
- GitHub Action improvements
- Documentation updates

## Guidelines

- All data contributions must include the raw scan results
- Feature classifications are determined by frequency analysis, not opinion
- Follow the existing code style
- One PR per feature/fix
